# 🎨 Banner / 主视觉设计草案

## 设计方向

项目关键词：
- QQ 群聊机器人
- API-first (阿里云/通义 API)
- 可选本地模型
- 开源自部署
- 群聊可唤起
- 现代技术感

---

## 方案一：「科技蓝 + 群聊气泡」⭐推荐

### 核心概念
以「消息气泡 + API 链路」为核心视觉，传达「QQ 群聊 + AI 云端能力」的产品定位。

### 标题
**AI Robot**
**QQ 群聊 AI 机器人**

### Slogan
**「让每个群都能调戏 AI」**

### 配色
```
主色: #1E88E5 (科技蓝)
辅色: #00ACC1 (青色)
强调: #FF6B6B (珊瑚红)
背景: #0D1B2A (深空蓝)
文字: #FFFFFF / #B0BEC5
```

### 视觉元素
- 左侧：QQ 消息气泡样式，内有 `@机器人` 文字
- 中间：数据流向箭头（消息 → API → 回复）
- 右侧：通义/阿里云 LOGO 元素
- 背景：渐变深蓝色 + 微弱电路纹理

### 动效建议
- 消息气泡轻微浮动
- 数据流向箭头脉冲闪烁

### AI 作图描述词
```
A modern tech banner for a QQ group chat AI bot project.
Dark blue gradient background (#0D1B2A to #1E3A5F).
Left side: QQ-style chat bubbles with Chinese text.
Center: glowing data flow arrows connecting to cloud icons.
Right side: stylized TongYi/Qwen logo element.
Style: clean, professional, developer-friendly.
Colors: blue (#1E88E5), cyan (#00ACC1), coral (#FF6B6B).
Modern tech aesthetic, not childish or marketing-like.
Text overlay: "AI Robot" in bold white font.
```

## 推荐选择

### ⭐ 最推荐：方案一
理由：
1. **群聊机器人定位清晰**：消息气泡 + @符号让人一眼看出是 QQ 机器人
2. **API-first 表达直观**：数据流向 + 云图标传达「云端 AI」概念
3. **适合中文开源项目**：科技蓝是中文技术项目最常见的主色调
4. **不过于幼稚**：保持专业感，不会让人以为是玩具项目

### 适用场景
- GitHub README 顶部
- 项目首页 Hero 区
- 社交媒体预览图
