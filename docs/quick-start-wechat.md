## 临时替代方案

如果你现在需要微信机器人能力，可以考虑：

1. **使用 QQ** - 当前版本已完整支持 QQ 机器人
   - 快速开始：[quick-start-qq.md](quick-start-qq.md)

2. **等待 v1.40** - 预计将支持微信接入

---

## 技术说明

微信接入需要：

- 微信公众号或企业微信
- valid puppet token
- 可能需要服务器公网 IP

具体技术方案将在 v1.40 版本确定。
