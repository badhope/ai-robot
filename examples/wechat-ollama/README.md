# WeChat + Ollama 示例

本示例展示如何使用 AI Robot 快速搭建一个基于微信和本地 Ollama 模型的聊天机器人。

## 快速开始

### 1. 启动 Ollama

```bash
ollama serve
```

### 2. 启动机器人

```bash
pnpm dev
```

### 3. 测试

- 私聊：直接发送消息给机器人
- 群聊：@机器人 或发送 /ai xxx

## 架构说明

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   微信     │────▶│  Server      │────▶│  Ollama    │
│  (Wechaty) │◀────│  (消息处理)  │◀────│  (本地模型) │
└─────────────┘     └──────────────┘     └─────────────┘
                           │
                           ▼
                    ┌──────────────┐
                    │   Session    │
                    │   (内存存储)  │
                    └──────────────┘
```

## 消息流程

1. 用户在微信发送消息
2. WechatAdapter 接收消息并转换为 ChatMessageEvent
3. Server 根据触发规则判断是否需要 AI 回复
4. 从 SessionStore 读取历史上下文
5. 拼装消息并调用 Ollama API
6. 将回复通过 WechatAdapter 发送回微信
